# Family Running Championships

Age-graded running rankings across generations.

## Update Process

1. Add race results to Excel
2. Run the staged workbook export
3. Validate and reconcile the complete bundle
4. Review and explicitly promote the validated bundle
5. Run repository tests
6. Commit through a feature branch and Pull Request

See [Workbook website export workflow](docs/workbook-export-workflow.md).
